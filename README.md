# AEPatcher

Движок патчей портирован из [исходного кода ApkEditor](https://github.com/timscriptov/ApkEditor "исходного кода ApkEditor").

Скачать готовый APK можно по ссылке: https://maximoff.su/apktool/AEPatcher.apk
